FRONT-END
Front-end development, also known as client-side development, involves creating the user interface (UI) and user experience (UX) of a website or web application.
It focuses on what users interact with directly in their browsers. Front-end developers use a combination of programming languages, frameworks, and tools to build visually appealing, responsive, and interactive web pages.
WE ARE MAINLY USED LANGUAGES ARE:
1.HTML
2.CSS
3.JS
BACK-END
Back-end development, also known as server-side development, focuses on the behind-the-scenes functionality of websites and web applications.
It involves managing the server, database, and application logic to ensure that the front-end (what users see and interact with) works seamlessly.
Back-end developers build and maintain the technology that powers the server, database, and application.
WE ARE MAINLY USED TECNOLOGIES ARE:
1.PYTHON
2.JAVA
3.JS(NODE.JS)
